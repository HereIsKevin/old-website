My Projects

Convoluta

A revolutionary ML-based solution to combat misinformation. Powered by TensorFlow and spaCy, Convoluta is built upon cutting edge technologies. Convoluta is set to launch around late 2021, with the API set to launch around early 2022.

neat

Fast and minimalistic bash prompt with infinite options for customizing. neat is extremely extensible, with a simple API for adding new components and plugins. With extensive caching, neat crushes Powerlevel10k and even starship at performance.

boreDOM

A lightweight and blazing-fast web components library built upon a rock-solid rendering system. By only diffing possible dynamic places, boreDOM does not need to diff the whole DOM tree, resulting in near zero-cost rendering. boreDOM is comparable to lit-html in performance and crushes React and Angular.

GeoBounce

Fun and simple, GeoBounce is a platformer game built by Boredom Games and is much like Geometry Dash and Boxel Rebound. With a custom-built game engine, GeoBounce sacrifices nothing for performance and the enjoyment of the player. Each level is a combination of challenges, fun, and much more, each more rewarding than the last.
